Conectores

Sistemas avanzados de empalmes mecánicos para barras de armaduras brindando soluciones para cada tipo de problema de conexión. Facilidad de utilización, fiabilidad, integridad estructural y mínima congestión. Están testados y comprobados, ofreciendo el método mas efectivo para conectar barras de armaduras y cumplir con las normas constructivas nacionales e internacionales. Los conectores mecánicos permiten una fiabilidad y consistencia que no se puede cumplir con un empalme.

CONECTORES SOLDABLES

Los conectores mas delgados en el mercado y su rosca cónica los convierte en los mas fiables y fáciles en su instalación, realizando un optimo balance coste/eficiencia. Los soldables o con rosca métrica se utilizan para realizar conexiones entre elementos en hormigón y en acero.

LENTON FORM SAVER

Es el Sistema ideal para eliminar barras salientes en caso de obras construidas por fases progresivas o en caso de aberturas temporales. Esta diseñado con rosca cónica LENTON y fabricado con una protección de la rosca unida a una placa, para una aplicación sencilla y segura al encofrado.

CADWELD

Se utilizan en el caso de que se exija una particular resistencia y solidez. Es el primer sistema de empalme mecánico jamás realizado, ofrece consistencia, versatilidad y flexibilidad. Es el Sistema mas utilizado a escala mundial para instalaciones nucleares.

LENTON INTERLOK

Es un Sistema de empalme para prefabricados por estar diseñado para dar continuidad estructural entre las barras de armaduras en elementos prefabricados. Tiene máxima eficiencia en construcciones prefabricadas, donde contribuye a eliminar las líneas antiestéticas entre los elementos estructurales con resultados arquitectónicamente muy validos.

ENTON QUICK WEDGE

Esta diseñado principalmente para aplicaciones de empalmes entre barras ya en obra y su aplicación es muy sencilla y rápida. Permite la eliminación de los costes y peligros de demolición de secciones de hormigón. Es un sistema extremadamente eficaz para ampliar viaductos, para mejorar estructuras en zonas sísmicas, para extensiones de pilares.

LENTON SPEED SLEEVE

Es la primera elección de las empresas para aplicaciones solo a compresión. Se ha diseñado para una rápida instalación y una inspección sencilla. Solamente utilizando extremidades de las barras escuadradas LENTON SPEED SLEEVE esta conforme a las normas internacionales de los empalmes solo a compresión.

LENTON TERMINATOR

Es un anclaje con rosca cónica que se pone a la extremidad de una barra de armadura para realizar un anclaje mas eficaz de un anclaje hecho a través de una barra doblada. Este Sistema simplifica el hueco de las armaduras, reduce la congestión y mejora la integridad estructural.

LENTON LOCK

Permiten una instalación sencilla in situ, puesto que no es necesario preparar, cortar, ni reducir los extremos de la barra. Los conectores pueden instalarse con una llave de tuercas estándar o con una llave de impacto, dependiendo del tamaño del conector. Las cabezas de los pernos se cortaran una vez alcanzada el apriete correcto de instalación, lo que permite una inspección visual.
